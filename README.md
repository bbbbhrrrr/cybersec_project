# 网络空间安全创新创业实践课程项目

## 项目概述

本仓库包含网络空间安全创新创业实践课程的项目作业，采用模块化的项目管理方式，每个项目独立开发和维护。

## 项目列表

### Project 1: SM4 加密算法性能优化
- **文件夹**: `project1-sm4-optimization/`
- **描述**: 实现SM4分组密码算法的多种性能优化技术，包括基础实现、T表优化和批处理优化
- **技术栈**: C语言、GCC编译器、AVX2指令集
- **状态**: ✅ 已完成
- **性能提升**: T表优化+46%，批处理优化+18%
